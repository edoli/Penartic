이 프로젝트는 gcode를 이용해서 그림을 그리는 프로젝트임. 내가 3d print가 남는게 있어서 그거를 이용해서 만들려고 하는것임. 펜 플로터에서도 비슷하게 작동할거 같음. usb 시리얼로 프린터와 연결해서 사용하는 방식.

Device(3d print 또는 펜 플로터)가 연결 되지 않는 상태에서도 작업이 가능 (출력은 안됨)
Device가 연결 안된 상태에서는 프린트 가능 사이즈를 모르기 때문에 최초에 기본값으로 설정되어 있고 수동으로 설정 할 수 있어야 함
Device를 연결하면 해당 디바이스의 정보를 가져와서 실제 프린트 가능 사이즈를 업데이트 함.

SVG 파일을 넣으면 알아서 gcode로 변환됨.
변환된 gcode가 렌더링 되어서 화면에 보임. 아래 좌우로로 슬라이더가 있어서 이 슬라이더를 조절하면 어떻게 그려지는지 과정을 볼 수 있음. prusa slicer나 bambu studio랑 비슷한 방식.

wgpu 렌더링을 이용해서 3d로 그리고 펜도 적당한 mesh로 만들어서 넣어줘. 그리고 슬라이더를 조절하면 3d 상에서 펜이 움직이는것을 확인하면서 그려지는 것을 확인할 수 있도록 해줘.

그려질때 연속적으로 그려져야 하는 부분은 XY 평면에서 계속 움직이게 해주고 중간에 띄워져야 하면 z축을 살짝 올려서 띄워지게 해줘.

프린트를 시작할때 Z축이 펜이 종에 맞닿아져 있다고 가정하고 z축을 살짝 올리고 XY 로 Home을 한번 해주고 나서 시작하게 해줘.

왼쪽에 사이드바에 설정 가능한 값들을 두고 3d 뷰포트에 프린트될 이미지와 펜을 3d 렌더링으로 표현해줘.
